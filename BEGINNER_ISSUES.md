# Beginner-Friendly Issues for GSSoC Contributors

## Issue Ideas (Create these in GitHub Issues)

### 1. [GOOD FIRST ISSUE] Improve Dashboard Loading States
**Description:** Add skeleton loading components to the dashboard while data is loading
**Files to modify:** `eyehealth/dashboard/src/components/LoadingSkeleton.tsx`
**Learning:** React components, CSS animations
**Time estimate:** 2-3 hours

### 2. [GOOD FIRST ISSUE] Add Keyboard Shortcuts to Extension
**Description:** Implement keyboard shortcuts (e.g., Ctrl+Shift+E to toggle extension)
**Files to modify:** `eyehealth/extension/content/overlay.ts`
**Learning:** Chrome extension APIs, event listeners
**Time estimate:** 3-4 hours

### 3. [GOOD FIRST ISSUE] Improve Error Messages in Popup
**Description:** Make error messages more user-friendly and actionable
**Files to modify:** `eyehealth/extension/popup/popup.tsx`
**Learning:** React state management, UX writing
**Time estimate:** 2 hours

### 4. [GOOD FIRST ISSUE] Add Dark Mode Toggle to Dashboard
**Description:** Add a toggle button to switch between light and dark themes
**Files to modify:** `eyehealth/dashboard/src/App.tsx`, `eyehealth/dashboard/src/index.css`
**Learning:** CSS variables, React hooks
**Time estimate:** 4-5 hours

### 5. [GOOD FIRST ISSUE] Create FAQ Section in README
**Description:** Add a comprehensive FAQ section to answer common questions
**Files to modify:** `README.md`
**Learning:** Technical writing, markdown formatting
**Time estimate:** 2-3 hours

### 6. [GOOD FIRST ISSUE] Add Tooltips to Dashboard Metrics
**Description:** Add hover tooltips explaining what each metric means
**Files to modify:** `eyehealth/dashboard/src/components/ScoreCard.tsx`
**Learning:** React components, CSS positioning
**Time estimate:** 3-4 hours

### 7. [GOOD FIRST ISSUE] Improve Extension Icon Design
**Description:** Create better icons for different extension states (active, inactive, warning)
**Files to modify:** `eyehealth/extension/icons/`
**Learning:** Icon design, SVG optimization
**Time estimate:** 2-3 hours

### 8. [GOOD FIRST ISSUE] Add Export/Import Settings Feature
**Description:** Allow users to export and import their EyeGuard settings
**Files to modify:** `eyehealth/extension/popup/popup.tsx`, `eyehealth/extension/db/schema.ts`
**Learning:** Browser APIs, JSON handling
**Time estimate:** 4-5 hours

### 9. [GOOD FIRST ISSUE] Create Quick Start Video Script
**Description:** Write a script for a 2-minute quick start video
**Files to modify:** Create `docs/video-script.md`
**Learning:** Technical communication, video planning
**Time estimate:** 2 hours

### 10. [GOOD FIRST ISSUE] Add Performance Metrics Dashboard
**Description:** Show extension performance metrics (CPU usage, memory, etc.)
**Files to modify:** `eyehealth/dashboard/src/components/PerformancePanel.tsx`
**Learning:** Browser performance APIs, React charts
**Time estimate:** 5-6 hours

## Instructions for Creating These Issues

1. Go to GitHub repository → Issues → New issue
2. Choose "Good First Issue" template
3. Fill in the details from above
4. Add appropriate labels: `good first issue`, `help wanted`
5. Assign to yourself or leave unassigned
6. Link relevant files and documentation
7. Set difficulty level and time estimate

## Tips for Success

- Provide clear, step-by-step instructions
- Include screenshots or mockups when possible
- Link to relevant documentation
- Offer help and guidance in the issue description
- Keep the scope manageable for beginners
